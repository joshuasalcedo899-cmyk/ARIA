ARIA: An Informative Augmented Reality Assistant for Front Desk Information Services

ARIA is a college capstone project developed as an Augmented Reality (AR) front desk assistant for STI College Calamba. The application is designed to provide students, visitors, and staff with quick, interactive, and accessible information without the need for a physical front desk personnel.

Using Augmented Reality technology, ARIA enhances the traditional information service experience by overlaying digital content in the real world. The assistant can guide users, answer common inquiries, and present institutional information in an engaging and modern way through a mobile device.
